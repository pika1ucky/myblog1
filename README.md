# React技术博客

使用React和Tailwind CSS构建的现代化技术博客。

## 功能特点

- 文章列表展示
- 分类筛选
- 搜索功能
- Markdown文章渲染
- 代码高亮
- 响应式设计

## 技术栈

- React 18
- TypeScript
- Tailwind CSS
- React Router
- React Markdown

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 运行测试
npm test

# 构建生产版本
npm run build
```

## 部署

本项目使用Vercel进行部署。
